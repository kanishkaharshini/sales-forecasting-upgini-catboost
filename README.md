# Store Item Demand Forecasting

## Overview
This project builds a sales forecasting regression model using CatBoost and Upgini feature enrichment.

## Dataset
Kaggle Store Item Demand Forecasting Challenge

## Workflow
- Data preprocessing
- Train/test split
- Feature enrichment with Upgini
- Model training using CatBoost Regressor
- Prediction generation
- Evaluation using SMAPE

## Technologies
- Python
- Pandas
- CatBoost
- Upgini

## Results
- Enrichment uplift observed during evaluation
- Forecasting performance measured using SMAPE
